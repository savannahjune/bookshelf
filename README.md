# Bookshelf

### Shows Top 40 Programming Books

#### To use the app locally:

```javascript
1.  clone the repo
2.  npm install
3.  npm run dev
4.  open localhost:3000
```

#### Development:

If you make changes, the app will rebuild and reload if there are no build issues.
